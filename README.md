# 計算カードアプリ

## URL

https://ichiko-git.github.io/cal-card-forchild/

## 概要

小学1年生向けの計算練習Webアプリです。学校の「計算カード」宿題の練習・代替として使用できます。たし算・ひき算・かけ算に対応し、小1でも迷わないシンプルなUIを実現しています。

## 対象ユーザー

- 小学1年生
- 計算カードの宿題を練習したい子ども
- スマートフォン・タブレットでの操作を想定

## 主な機能

### 1. モード選択
- **たしざん**: くりあがりなし／ありから選択
- **ひきざん**: くりさがりなし／ありから選択
- **かけざん**: 2のだん、3のだん、ぜんぶまぜるから選択

### 2. 設定選択
- **出題順**: ならびじゅん（固定順）またはランダム
- モードに応じた詳細設定を選択

### 3. 問題画面
- **たし算・ひき算**: 選択式（答えの候補ボタンから選択）
- **かけ算**: 入力式（数字ボタンで入力してOKボタンで決定）
- 進捗表示（何問目 / 全問数）
- 正解・不正解の即時フィードバック
  - 正解時: ⭐ せいかい！表示＋効果音
  - 不正解時: ✕ ざんねん…表示

### 4. 結果画面
- 正解数（全問数に対する正解数）
- 間違えた数（一度でも間違えた問題の数）
- かかった時間（分・秒）
- がんばりメッセージ
  - 全問正解かつ一度も間違えなかった場合: 「ぜんもんせいかい！」「すごいね！」
  - 80%以上正解: 「とってもよくできました！」
  - その他: 「よくがんばったね！」

## 問題の仕様

### たし算
- 範囲: 1〜9 + 1〜9
- くりあがりなし: 答えが10以下
- くりあがりあり: 答えが11以上

### ひき算
- **くりさがりなし**: 9-8までの問題、答えが0でない
- **くりさがりあり**: 1〜18 - 1〜9（答えが0以上）

### かけ算
- **2のだん**: 2×1〜2×9（9問）
- **3のだん**: 3×1〜3×9（9問）
- **ぜんぶまぜる**: 2のだんと3のだんを合わせた18問

## 特徴

- **シンプルなUI**: 文字説明は最小限、色と配置で理解させる
- **1画面1目的**: 迷わず操作できる
- **固定問題を全問出題**: ランダム出題はしない（順番のみ切り替え可能）
- **子どもを焦らせない**: タイムによる評価や煽り表示はしない
- **即時フィードバック**: 正解・不正解は即座に表示
- **間違いの記録**: 一度でも間違えた問題は正解数にカウントしない

## 使用技術

- **HTML5**: 基本的な構造
- **CSS3**: レスポンシブデザイン、アニメーション
- **React 18**: UIコンポーネントと状態管理
- **JavaScript (ES6+)**: 問題生成ロジック、時間計測
- **Babel Standalone**: JSX変換（CDN経由）

- スマートフォン・タブレット対応
- 外部通信なし（オフライン動作可能）

## 開発メモ

- ReactはCDN経由で読み込んでいます（ビルドプロセス不要）

- 問題データは実行時に生成されます
