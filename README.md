# 計算カードゲーム

## URL

https://ichiko-git.github.io/cal-card-forchild/

## 概要

小学校低学年向けの計算練習ゲームです。足し算・引き算・かけ算に対応し、問題の出し方（じゅんばん／ランダム）や、足し算の「くりあがりの有無」を選んでからスタートします。正解時には効果音が鳴り、全問終了後にかかった時間を表示します。
このゲームはスマートフォンで操作することを想定されるため、スマートフォン向け UI にしています。

## 機能

- **演算の選択**
  - たし算（くりあがりなし／あり）
  - ひき算
  - かけ算
- **出題順序の選択**
  - じゅんばん（規則順）
  - ランダム
- **回答 UI**
  - たし算・ひき算: 候補ボタンから選択
    - くりあがりなし: 1〜10
    - くりあがりあり: 11〜18
  - かけ算: 電卓風の数字ボタンで入力し「けってい」を押す
- **進捗表示**
  - プログレスバーで全体の進み具合を可視化
- **正解フィードバック**
  - 正解で「○ せいかい！」表示＋効果音再生
- **結果表示**
  - 全問終了後、合計所要時間（分・秒）を表示

## 使用技術

- HTML
- CSS
  - ラジオ風 UI、ボタン、プログレスバーのスタイル
- JavaScript
  - DOM 操作、イベントハンドリング
  - 問題生成（足し算・引き算・かけ算）
  - 状態管理（問題配列、現在インデックス、開始・終了時刻）
  - 効果音の再生制御

## 学んだこと

- ユーザー入力に応じた UI の出し分け（演算種別に応じて回答 UI を変更）
- 子ども向け UI/UX（大きなボタン、電卓レイアウト、結果の即時フィードバック）
- 時間計測と進捗可視化（プログレスバー、経過時間表示）
- 配列シャッフルやガード条件を用いた問題生成ロジックの整理
